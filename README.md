#  google-serp-parser

Парсер результатов поиска Google.

Этот проект анализирует сохраненную HTML-страницу результатов поиска Google по запросу "get taxi".

##  Примечания

- Анализ выполняется с использованием **регулярных выражений**.
- Этот подход является ненадежным и предназначен **только для демонстрации**.
- В **production** для выполнения данной задачи лучше использовать **DOM-парсер**, так как структура Google SERP может изменяться.

##  Структура проекта

```
project/

├── output/

├── package.json
├── tsconfig.json

```

##  Как запустить

1. Сохранение HTML-кода результатов поиска Google в `input/google.html`
1. Установка зависимостей: `npm install`
1. Сборка: `npm run build`
1. Запуск: `npm start`

##  Вывод

Результаты сохраняются в `output/result.csv`

## Чтобы я улучшил в дальнейшем развитии проекта:

- Расширенная обработка ошибок:
  - проверка существования входного HTML-файла;
  - проверка корректности структуры страницы;
  - защита от пустого или повреждённого HTML.

- Улучшение нормализации данных:

- Логирование этапов выполнения (чтение файла, парсинг, количество результатов, ошибки).
- Добавить unit-тесты для парсера.
